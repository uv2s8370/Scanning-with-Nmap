# Network Scanner Automation Tool

A lightweight, automated Bash script designed to streamline network reconnaissance and port scanning using Nmap. This tool simplifies host discovery and helps bypass strict local firewalls using optimal scanning profiles.

## 🚀 Features

* *Fast Scanning:* Automatically targets the most vital network ports to save time.
* *IP Range Support:* Easily handles individual targets.

## 📋 Prerequisites

Before running this script, ensure you have the following installed on your Unix-like system (Kali Linux, Ubuntu, macOS):

* *Bash Shell* (Standard on most Linux distros)
* *Nmap* (Install via sudo apt install nmap)

## Grant Execution Permissions
By default, Linux restricts new scripts from running. Give the script execution rights with:

chmod +x scan.sh
